O ReactJS divide a interface de usuário em pedaços de código isolados e reutilizáveis, conhecidos como componentes. Os componentes do React funcionam de maneira semelhante às funções JavaScript, pois aceitam entradas arbitrárias chamadas propriedades ou props. Podem ser repetidos com informações diferentes. É uma função que retorna HTML

Os elementos React retornados determinam como a interface do usuário ficará no lado do cliente. Aqui está um exemplo de um componente de função que retorna um elemento React:

function Welcome(props) {
  return h1>Olá, {props.name}/h1>;
}
É possível ter quantos componentes forem necessários sem sobrecarregar seu código.
